# Atlassian Case Study: Improving Sprint Efficiency

## Overview
This project analyzes simulated Jira ticket data to identify inefficiencies in sprint performance, ticket resolution, and workload distribution.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

## Key Insights
- High-priority tickets take longer to resolve
- Reopened tickets increase delays
- Workload across engineers is uneven
- Some teams are slower than others

## Project Structure
- notebooks/ → analysis notebook
- data/ → dataset
